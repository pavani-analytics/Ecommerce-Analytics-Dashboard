# E-Commerce Analytics Dashboard

## Project Overview

This project analyzes e-commerce sales, customers, products, payments, reviews, sellers, and delivery performance using Excel, Power BI, Tableau, and SQL. The project focuses on identifying sales trends, customer behavior, payment preferences, product performance, seller performance, and delivery efficiency through interactive dashboards and analytical reporting.

## Dataset

📊 Dataset Files: https://drive.google.com/file/d/1oB-3LhKaw7xfD4lZ3AYzsP2hbkNFVl2H/view?usp=sharing

The project utilizes the following datasets:

* Customers
* Geolocation
* Products
* Order Items
* Order Payments
* Order Reviews
* Orders
* Sellers
* Product Category Translation
